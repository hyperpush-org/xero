lanaai inside
05 — integration
Where LanaAI plugs in

LanaAI becomes the on-chain intelligence every run calls.

We don't want to wrap LanaAI. We want to give it a home inside the workflow builders already use — so a conversational explorer turns into the agent's working memory.

01

Tx inspector + decoder

After simulation or failure, Xero hands LanaAI the signature, logs, IDL, writable accounts, and persona context — and gets a plain-English trace back.

02

Audit + exploit replay

Before a risky deploy, the audit engine asks LanaAI for similar attack patterns, related account flows, and a short replay note humans can skim in 30 seconds.

03

Deploy-safety gate

Program upgrade checks query LanaAI for authority history, recent program activity, suspicious token flows, and IDL drift context — all before the button goes green.

04

Persona + scenario seeding

A builder describes a market, a wallet, an incident. Xero turns it into funded personas and a fork-ready scenario, with LanaAI filling in the on-chain context.

The upshot for Helius — every Xero Solana session becomes a LanaAI power session. Human prompts turn into agent loops. Agent loops multiply query volume.
